    <h1>Settings overview</h1>
    <p>Click on the <em class="fas fa-cogs"> </em>Game Settings tab then under the Game Settings heading click on <em class="fas fa-cogs"> </em>Configure Settings.</p>
    <p>Click on <em class="fas fa-cogs"> </em>System Settings</p>
    <ul>
        <li><em>Variant/Optional Rules</em> - Here you can turn on individual Pulp Cthulhu rules and other optional rules</li>
        <li><em>Initiative Settings</em> - Additional settings for optional initiative rule</li>
        <li><em>Roll Settings</em> - Default options for rolls</li>
        <li><em>Chat Cards Settings</em> - Configure chat messages</li>
        <li><em>Scene Settings</em> - Scene Settings</li>
        <li><em>Game Artwork Settings</em> - This allows you to set a custom pause icon and message</li>
        <li><em>Sheet Settings</em> - This allows you to change character sheet settings and optional CSS</li>
        <li><em>Weapon Settings</em> - Weapon Settings</li>
        <li><em>Developer And Debug Settings</em> - These settings can break your world when new updates are released so only use them on test worlds</li>
        <li><em>Roll Table Settings</em> - When sanity rolls are made the system can automatically roll for a bout of madness. You can see example roll tables in the Sanity Roll Table compendiums</li>
    </ul>
    <h1>Call of Cthulhu Scene Menu</h1>
    <p>To access this menu you will need to have an active scene which can be created in the <em class="fas fa-map"> </em>Scenes Directory. These options are only available to the Keeper.</p>
    <ul>
        <li>
            <p><em>Keeper's tools</em></p>
            <ul>
                <li><em>Development phase</em>: When enabled, players can make improvement rolls for their marked skills.</li>
                <li><em>Character creation mode</em>: When enabled, players can distribute points among their skills.</li>
                <li><em>XP gain</em>: When enabled, a skill will be marked for improvement after a successful check.</li>
                <li><em>Send a decoy roll to players</em>: When clicked, players will see a fake GM private roll.</li>
                <li><em>Start Rest</em>: When click, pick characters to perform a rest and roll for XP gains.</li>
            </ul>
        </li>
        <li><em>Roll !</em>: Used to roll 1d100 with a threshold, difficulty and bonus or penalty dice.</li>
        <li><em>Create link</em>: Create a roll link for players to click</li>
    </ul>
    <h1>Keyboard and mouse shortcuts</h1>
    <p>There are many elements in the sheets that trigger a dice roll when clicked. Usually a dialog is shown to prompt the user for a difficulty and a possible bonus or penalty. This behavior is modified with the following controls:</p>
    <ul>
        <li>Right click on any rollable element to include it in an opposed roll. As long as the card is open, all rolls made with a right click will be added to the opposed roll.</li>
        <li>Alt + Right click on any rollable element to include it in a combined roll.</li>
        <li>Shift + Left click on a rollable element will make a roll without asking for difficulty or bonus/penalty.</li>
        <li>Ctrl + Left click on a rollable element will create a roll request. Only available for the GM.</li>
        <li>Alt + Left click on sanity will prompt the player for minimum and maximum sanity loss.</li>
    </ul>
