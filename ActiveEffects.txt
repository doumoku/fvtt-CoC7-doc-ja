<div class="coc7overview">
    <h1>効果</h1>
    <p>このシステムはアクティブ効果の生成できます。アクティブ効果は、アクターの能力値、副能力値、技能を修正します。効果は@UUID[.9YesdNtaElDQaaaZ]{リンク作成ツール}を使用することで、@UUID[.7sSX5w3psDau2CDn]{リンク}として作成できます。また、キャラクターシート内の<em class="game-icon game-icon-aura"> </em> ボタンを押すことでも作れます。</p>
    <h2>効果タブ</h2>
    <p>効果はキャラクターシートの「効果タブ」に表示されます。</p>
    <img src="systems/CoC7/assets/manual/effects/effects-tab.webp" alt="効果タブ">
    <p>探索者用の効果は４つのカテゴリに分けられます：</p>
    <ul>
        <li>状態：システムから生成され、使用される効果です（重傷、伏せ、発狂など…）。 これらの効果は変更を含みません。</li>
        <li>一時的：持続時間のある効果です。</li>
        <li>受動：恒久的な効果です。</li>
        <li>無効：無効化された効果です。</li>
    </ul>
    <p>NPCとクリーチャーシートは、有効と無効のカテゴリが２つ表示されます。効果が無効でない間、アクターに修正が適用されます。</p>
    <h2>効果の作成</h2>
    <p>追加ボタンをクリックすることで効果を作成し、効果画面が開きます。この画面には３つのタブがあります。</p>
    <h3>詳細タブ</h3>
    <img src="systems/CoC7/assets/manual/effects/details-tab.webp" alt="Details tab">
    <p>効果の名称を設定できます。</p>
    <h3>持続時間タブ</h3>
    <img src="systems/CoC7/assets/manual/effects/duration-tab.webp" alt="Duration tab">
    <h3>効果タブ</h3>
    <img src="systems/CoC7/assets/manual/effects/changes-tab.webp" alt="Changes tab">
    <p>最後のタブにはキャラクターシートに行った変更点がすべて含まれます。</p>
    <h2>変更</h2>
    <p>変更点を含む効果です。変更点はそれぞれ対応するシステムパスによって指定しなければいけません。指定可能な変更点は以下の通りです：</p>
    <ul>
        <li>
            <p>能力値：</p>
            <ul>
                <li>
                    <p>STR：</p>
                    <ul>
                        <li>system.characteristics.str.value</li>
                        <li>system.characteristics.str.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>CON：</p>
                    <ul>
                        <li>system.characteristics.con.value</li>
                        <li>system.characteristics.con.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>SIZ：</p>
                    <ul>
                        <li>system.characteristics.siz.value</li>
                        <li>system.characteristics.siz.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>DEX：</p>
                    <ul>
                        <li>system.characteristics.dex.value</li>
                        <li>system.characteristics.dex.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>APP：</p>
                    <ul>
                        <li>system.characteristics.app.value</li>
                        <li>system.characteristics.app.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>INT：</p>
                    <ul>
                        <li>system.characteristics.int.value</li>
                        <li>system.characteristics.int.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>POW：</p>
                    <ul>
                        <li>system.characteristics.pow.value</li>
                        <li>system.characteristics.pow.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>EDU：</p>
                    <ul>
                        <li>system.characteristics.edu.value</li>
                        <li>system.characteristics.edu.bonusDice</li>
                    </ul>
                </li>
            </ul>
        </li>
        <li>
            <p>副能力値：</p>
            <ul>
                <li>
                    <p>幸運：</p>
                    <ul>
                        <li>system.attribs.lck.value</li>
                        <li>system.attribs.lck.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>正気度：</p>
                    <ul>
                        <li>system.attribs.san.value</li>
                        <li>system.attribs.san.bonusDice</li>
                    </ul>
                </li>
                <li>
                    <p>MOV（移動率）：</p>
                    <ul>
                        <li>system.attribs.mov.value</li>
                    </ul>
                </li>
                <li>
                    <p>ビルド：</p>
                    <ul>
                        <li>system.attribs.build.value</li>
                    </ul>
                </li>
                <li>
                    <p>ダメージ・ボーナス：</p>
                    <ul>
                        <li>system.attribs.db.value</li>
                    </ul>
                </li>
                <li>
                    <p>装甲：</p>
                    <ul>
                        <li>system.attribs.armor.value</li>
                    </ul>
                </li>
            </ul>
        </li>
        <li>
            <p>副能力値は、その最大値のみを修正してください。これらの変更点は他の修正が完了した後に適用されます。変更する副能力値が自動モードの場合、変更前の能力値で再計算されてから値が修正されます。</p>
            <ul>
                <li>
                    <p>耐久力：</p>
                    <ul>
                        <li>system.attribs.hp.max</li>

                </li>

                    <ul>

                </li>
            </ul>
        </li>
        <li>
            <p>技能の指定には、完全な名称を使用してください。技能は大文字小文字を区別して判別されています！</p>
            <ul>
                <li>
                    <p>〈魅惑〉</p>
                    <ul>
